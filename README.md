# Moneyball

This is a play on the moneypuck website. It will become a flask website but right now, you can run a script to generate a report on any MLB team's last game. It will give you simple advanced statistical analysis for that game.

## Quick Start

Ensure Python 3.13 is installed on your machine, other versions may work, but are not supported.

Install the following libraries to set up:

```
pip install matplotlib
pip install pandas
pip install pybaseball
pip install MLB-statsapi
```

Then run `python get_report.py` in the home directory.

Happy analysis! :)
